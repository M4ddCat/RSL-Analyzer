# RSL Analyzer

Поддержка языка программирования RSL (Rsd Script Language) для Visual Studio Code. Это расширение обеспечивает высокоточную подсветку синтаксиса и глубокое понимание структуры кода благодаря мощному парсеру на базе Tree-sitter.

## Основные возможности

- **Умная подсветка синтаксиса**: Полная поддержка всех конструкций языка, включая макросы, классы и переменные.
- **Продвинутый парсинг RSDCommand**: Корректная обработка многострочных SQL-запросов .
- **Поддержка макросов**: Глубокая интеграция с встроенными макросами и системными переменными.

## Технологии

Расширение использует парсер **Tree-sitter**, что гарантирует:
- Высокую скорость работы даже на очень больших файлах.
- Устойчивость к синтаксическим ошибкам (подсветка не «ломается» при редактировании).
- Точную идентификацию областей видимости (scope).
